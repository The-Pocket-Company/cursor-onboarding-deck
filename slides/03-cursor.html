Cursor 先當成「會陪你寫程式的編輯器」

你已經有帳號，所以第一步不是學完所有設定，而是知道它每天會幫你做哪三件事。

1

開一個專案資料夾

所有程式碼、設定、文件都放在同一個 workspace。

2

用 AI 問「下一步」

像問助教一樣：請它解釋檔案、修 bug、整理步驟。

3

把成果存進 Git

等一下接 GitHub 後，你的專案就能保存版本。
