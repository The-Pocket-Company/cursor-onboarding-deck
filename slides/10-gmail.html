如果要用 GCP，Gmail 就是進雲端的鑰匙

GCP 是 Google Cloud Platform。通常你會用 Google 帳號登入，很多初學者最簡單就是準備 Gmail。

1

登入 Google Cloud Console

建立專案、啟用服務、管理帳單都在這裡。

2

建立 GCP Project

一個 project 像一個雲端工作區，裡面放 Cloud Run、Cloud SQL、Firestore。

3

管理權限與服務

未來也會用這個帳號決定誰能部署、誰能看資料。
