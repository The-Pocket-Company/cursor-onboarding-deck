connect to cursor

把 GitHub 連進 Cursor，專案才有去處

連接後，你可以 clone 別人的 repository，也可以把自己的專案 publish 到 GitHub。

git config --global user.name "你的名字"

git config --global user.email "你的 email"

這兩行只是告訴 Git「誰做了這次修改」。帳號登入則通常在 Cursor / GitHub 授權畫面完成。
