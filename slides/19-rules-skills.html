customize cursor

用 Rules 與 Skills 客製化 Cursor

前面教的原則，可以變成固定檔案餵給 Cursor，每次自動遵守。兩種做法：

rule · 自動套用

Rule 規則

固定的行為準則，可設成每次都自動套用。把「想清楚再寫、最小改動」這類原則寫進去。

.cursor/rules/<名稱>.mdc
skill · 需要時叫用

Skill 技能

需要時才被叫用的能力包。像這份簡報，就是用 huashu-design 這個 skill 做出來的。

.cursor/skills/<名稱>/SKILL.md

資料夾結構（放在專案根目錄）

你的專案/
└─ .cursor/
   ├─ rules/
   │  └─ ai-coding-guidelines.mdc   ← Rule
   └─ skills/
      └─ ai-coding-guidelines/
         └─ SKILL.md   ← Skill

1在專案根目錄建立 .cursor 資料夾與上面的子資料夾。

2新增 .mdc 與 SKILL.md 檔案。

3貼上內容、存檔，Cursor 立即生效。下一頁可直接複製。
