work modes

兩種模式：先想清楚，再動手

Cursor 有兩種工作方式。學會在對的時機切換，是把 AI Coding 做準的第一步。

think first

Plan 模式

唯讀模式，先一起把做法想清楚。Cursor 會問問題、提出計畫，但不會改你的檔案。

適合什麼時候

需求比較大，或有好幾種做法

想先釐清方向再動手

需要一份能先確認的 Plan

execute

Agent 模式

執行模式，會實際讀寫檔案、跑指令，把任務直接做出來。

適合什麼時候

方向已經很清楚

小修改、單一明確任務

直接 build，馬上看結果

心法

複雜任務先用 Plan 想清楚，方向定了再切 Agent 開始 build。
